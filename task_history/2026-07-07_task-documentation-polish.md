## Исходное задание

Довести документацию Prompt Review до максимально завершённого состояния перед получением итогового вердикта по модулю.

**Цель:** Чтобы проект производил впечатление практически завершённого инженерного продукта.

**Общий принцип:**
- Не создавать новые документы без объективной необходимости
- Не расширять методологию APL
- Не проектировать новые стандарты
- Основная задача — довести до высокого качества уже существующую документацию
